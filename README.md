---

## 📂 Como rodar o projeto localmente

### 1. Clone o repositório

```bash
git clone https://github.com/maikogabriel/portfolio.git
```

### 2. Acesse a pasta do projeto

```bash
cd portfolio
```

### 3. Instale as dependências

```bash
npm install
```

### 4. Rode o projeto

```bash
npm run dev
```

O projeto estará disponível em:

```bash
http://localhost:5173
```
